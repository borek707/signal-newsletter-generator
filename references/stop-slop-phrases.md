# Stop-Slop Reference: Phrases to Remove

AI writing has predictable patterns. This reference lists banned phrases that make text sound generated.

## Throat-Clearing Openers

Cut these entirely:

- "Here's what you need to know"
- "In this article, we'll explore"
- "Let's dive in"
- "It's worth noting that"
- "Before we begin"
- "First, let's understand"
- "As we can see"
- "What this means is"
- "The bottom line is"
- "At the end of the day"
- "When all is said and done"

## Emphasis Crutches

These words add nothing. Delete them:

- crucial
- critical
- essential
- key
- vital
- game-changer
- revolutionary
- groundbreaking
- innovative
- cutting-edge
- state-of-the-art
- seamless
- robust
- scalable (unless describing specific metrics)
- powerful

## Business Jargon

Replace with specific language:

| Avoid | Use Instead |
|-------|-------------|
| leverage | use |
| synergy | [delete or be specific] |
| ecosystem | community/platform/market |
| paradigm shift | change |
| best practices | [name the specific practice] |
| deep dive | analysis |
| moving forward | [delete] |
| at scale | [name the scale: "with 50+ brokers"] |
| end-to-end | complete |
| holistic | comprehensive |
| actionable | useful |
| learnings | lessons |

## All Adverbs

Remove all adverbs, especially:

- importantly
- significantly
- effectively
- efficiently
- ultimately
- basically
- essentially
- actually
- certainly
- definitely
- probably
- potentially
- increasingly
- increasingly important

Exception: Technical adverbs that add precision (e.g., "latency increases linearly")

## Vague Declaratives

Replace with specifics:

| Vague | Specific |
|-------|----------|
| The reasons are structural | [Name the structure] |
| The implications are significant | [Name the implication] |
| This represents a major shift | [Describe the shift] |
| There are several factors | [List the factors] |
| Various approaches exist | [Name the approaches] |

## Meta-Commentary

Delete entirely:

- "As mentioned earlier"
- "As discussed above"
- "In conclusion"
- "To summarize"
- "To wrap up"
- "The takeaway here is"
- "What this shows us"
- "This demonstrates that"

## Transition Phrases to Avoid

- "Moreover,"
- "Furthermore,"
- "Additionally,"
- "In addition,"
- "Not only... but also"
- "On the other hand,"
- "It's important to note"
- "Another key point"
